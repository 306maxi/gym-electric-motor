Dashboard¶

In the dashboard, the environment states are visualized during an episode. The plot is updated simultaneously to the run time of the control. The input voltage is drawn in magenta and the other state variables are plotted in blue. References are shown in green. Furthermore, nominal limits (dotted-yellow) and technical limits (dashed-red) are depicted. The plots can be used to analyze the behaviour of the controller.

Constructor of the dashboard.

Parameters

  • variables – Names of the variables that shall be shown on the dashboard

  • tau – system time constant

  • low – Array of lowest possible values for the variables in the same order as the variables

  • high – Array of highest possible values for the variables in the same order as the variables

  • episode_length – episode length of the system

  • safety_margin – Ratio between maximal and nominal power of the motor parameters.

  • with_ref – Array of Booleans indicating if the variables have a reference that has to be plotted

  • update_cycle – Number of steps after which the dashboard shall be updated. | Updating every cycle leads to a very low speed.
